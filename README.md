Natours - Travel app

Main Features: 
  1. Display All Tours
  2. Display Tour detail screen with Map
  3. Login
  4. Signup
  5. User Settings
  6. User reviews
  7. User bookings
  
Database: Mongodb - stored on ATLAS MongoDB (AWS)
Backend:  NodeJS & Express Restful API using Mongoose. 
          jsonwebtoken, promisify, and crypto for security.
Frontend: Angular 9
